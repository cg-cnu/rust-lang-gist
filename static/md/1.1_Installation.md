# rustup

* Install 

    ```bash
    $ curl https://sh.rustup.rs -sSf | sh
    ```

* Updating

    ```bash
    $ rustup update
    ```

* Uninstall

    ```bash
    $ rustup self uninstall
    ```

* Version

    ```bash
    $ rustc --version
    ```

* Local documentation

    ```bash
    $ rustup doc
    ```

* rustc - is the compiler

* rustup - helps install, update and uninstall rustc
