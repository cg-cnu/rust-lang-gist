# rustup

* Install 

```bash
$ curl https://sh.rustup.rs -sSf | sh
```

* Updating

```bash
$ rustup update
```

* Uninstall

```bash
$ rustup self uninstall
```

* Troubleshoot

```bash
$ rustc --version
```
